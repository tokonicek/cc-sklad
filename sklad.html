📖 Návod k použití
Rychlý start
Krok 1
Přidej produkty
Jdi do Produkty → + Nový produkt. Kód produktu je povinný, EAN a čárový kód jsou volitelné. Nebo importuj hromadně přes Import CSV.
Krok 2
Nastav objekty a pracovníky
V sekci Objekty & Pracovníci přidej objekty (VN, PV, Čáslavská…) a pracovníky. Pak se budou zobrazovat v dropdownech při výdeji.
Krok 3
Příjem zboží
Každý příjem vytvoří novou FIFO dávku s vlastní cenou a datem. Pro hromadný příjem z faktury použij Příjem přes fakturu — zadáš číslo faktury, přidáš produkty, potvrdíš vše najednou a lze exportovat PDF.
Krok 4
Výdej zboží
Výdej odpisuje ze skladu metodou FIFO (nejstarší dávka první). Pro hromadný výdej použij Výdej přes výdejku — přidáš produkty, potvrdíš, vygeneruje se PDF výdejka.
Krok 5
Inventura
V sekci Inventura klikni Načíst stavy. Oprav skutečné množství u každého produktu. Při potvrzení se automaticky zaúčtují rozdíly jako pohyby typu Inventura a vytvoří se audit záznam.
Krok 6
OneDrive sync
Data se automaticky ukládají na OneDrive po každé změně (debounce 3s). Pro nastavení jdi do OneDrive & Sync a vlož Client ID z Azure. Zálohy se vytvářejí automaticky do složky /ČČ s.r.o./Sklad/Zaloha/.
Import produktů z CSV
CSV soubor musí mít první řádek jako hlavičku. Povinný sloupec: kod. Volitelné: nazev, ean, barcode, kategorie, jednotka, min_mnozstvi, umisteni, url, poznamka, vendor_id
Příklad CSV:
kod;nazev;ean;kategorie;jednotka;min_mnozstvi
UKL001;Jar 1l;8710847911750;Chemie;ks;5
UKL002;Čisticí prášek;8710847911751;Chemie;kg;2

📖 Průvodce: Registrace Azure aplikace (~10 min, zdarma)
Krok 1
Přihlás se na Azure
Jdi na portal.azure.com → vyhledej App registrations.
Krok 2
Nová registrace
Klikni + New registration. Název: CC Sklad. Typ účtů: Personal Microsoft accounts. Redirect URI: typ SPA, hodnota http://localhost. Klikni Register.
Krok 3
Zkopíruj Client ID
V sekci Essentials zkopíruj Application (client) ID a vlož do pole výše.
Krok 4
API Permissions
API permissions → + Add → Microsoft Graph → Delegated → Files.ReadWrite + offline_access.
🔄 Aktualizace cen z OneDrive
Načte soubor ceny.json z OneDrive (cestu nastav v poli výše nebo v sekci Nastavení). Formát: {"KOD": 99.90, ...} nebo seznam [{"kod":"...", "cena": 99.90}]. Ceny se uloží jako ruční cena u produktů bez aktivní dávky.
